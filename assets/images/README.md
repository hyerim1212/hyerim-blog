# 사진 저장 위치

프로젝트 사진은 이 폴더 아래에 저장합니다.

추천 구조:

```txt
assets/images/
  simulation/
    week-1-model.png
    week-2-result-graph.png
  robot-arm/
    week-1-sketch.jpg
    week-2-parts.jpg
  study/
    python-note.png
```

Markdown 글에서 사진을 넣을 때는 다음처럼 작성합니다.

```md
![사진 설명](./assets/images/simulation/week-1-model.png)
```
